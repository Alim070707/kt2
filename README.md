Задание «КТ-2 Создать интерактивное приложение»

- Создайте новый проект Windows Forms в Visual Studio.
- Добавьте на форму несколько элементов управления, таких как кнопки, текстовые поля и метки.
- Добавьте обработчики событий для каждого элемента управления. Например, для кнопки можно добавить обработчик события Click, который будет выполнять определенные действия при нажатии.
- Измените свойства элементов управления в обработчиках событий. Например, для метки можно изменить текст в обработчике события Click кнопки.
